# Makefile

### Clone Project

```bash
$ git clone https://github.com/marcomansomm/Atividade01.git
```

### Make

```bash
$ make
$ gcc zumbi -o bin 
```

### Running

```bash
$ make run
$ ./zumbi
```

### Cleaning

```bash
$ make clean
$ rm bin
```
